Linux 🐧

Ein quelloffenes, extrem stabiles Betriebssystem. In der IT-Welt ist Linux das Rückgrat für Server, Cloud-Infrastrukturen und Netzwerke. Wir behandeln hier Rechteverwaltung, Shell-Skripte und Cronjobs.

Java ☕

Eine objektorientierte Programmiersprache, die nach dem Prinzip "Write Once, Run Anywhere" funktioniert. Verwendet für die Entwicklung von Unternehmensanwendungen und Web-Backends.

C# 💠

Eine moderne, objektorientierte Programmiersprache von Microsoft. Sie ist besonders stark in der Entwicklung von Windows-Anwendungen, Unternehmenssoftware und Backend-Services.

Netzwerke 🌐

Die Infrastruktur der IT. Hier geht es um Datenübertragung, Routing, Switching und Netzwerksicherheit. Kernkonzepte sind das OSI-Modell und TCP/IP-Protokolle.

Git 🐙

Das Standard-Versionskontrollsystem der Industrie. Es ermöglicht Teams, Code-Änderungen zu verfolgen, asynchron zusammenzuarbeiten und Branches sicher zusammenzuführen.

HTML & CSS 🎨

Die grundlegenden Bausteine des Webs. HTML strukturiert die Inhalte semantisch, während CSS für das visuelle Layout, Farben und Animationen sorgt.
