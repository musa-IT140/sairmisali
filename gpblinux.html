Linux 🐧

Ein quelloffenes, extrem stabiles Betriebssystem. In der IT-Welt ist Linux das Rückgrat für Server, Cloud-Infrastrukturen und Netzwerke.

Java ☕

Eine objektorientierte Programmiersprache, die nach dem Prinzip "Write Once, Run Anywhere" funktioniert. Verwendet für die Entwicklung von Unternehmensanwendungen.

C# 💠

Eine moderne, objektorientierte Programmiersprache von Microsoft. Besonders stark in der Entwicklung von Windows-Anwendungen und Backend-Services.

Netzwerke 🌐

Die Infrastruktur der IT. Hier geht es um Datenübertragung, Routing, Switching und Netzwerksicherheit basierend auf dem OSI-Modell.

Git 🐙

Das Standard-Versionskontrollsystem der Industrie. Es ermöglicht Teams, Code-Änderungen zu verfolgen und asynchron zusammenzuarbeiten.
